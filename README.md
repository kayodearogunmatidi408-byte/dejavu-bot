# 🧠 DejaVu Bot

A Telegram bot that saves messages and reminds you of them 1 year later!

## Commands
- `/save` - Save a message (reply to it)
- `/random` - Get random memory
- `/stats` - View top savers

## Features
- 📌 Inline save buttons
- 📅 Daily throwbacks at 9 AM
- 🎲 Random memory retrieval
